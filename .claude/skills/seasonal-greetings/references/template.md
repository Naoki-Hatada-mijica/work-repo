# 文面テンプレート（夏版・冬版）

差出人は「株式会社mijica 畑田」。宛名は `株式会社●●／●●様` を差し込み欄とする。

## 夏版（お中元・夏期休業）

**件名：【株式会社mijica】夏のご挨拶ならびに夏期休業のご案内**

```
株式会社●●
●●様

お世話になっております。mijicaの●●です。

平素は格別のご厚情を賜り、誠にありがとうございます。
日頃より弊社にご用命いただき、皆様の事業に携わらせていただけますこと、心より御礼申し上げます。

さて、誠に勝手ながら、弊社の夏期休業につきまして、下記の通りご案内申し上げます。

――――――――――――――
【夏期休業について】
・休業前最終営業日：{最終営業日}
・営業開始日　　　：{営業開始日}
・休業期間　　　　：{休業開始}～{休業終了}

※休業期間中に緊急のご用件がございました際は、
　各営業担当までご連絡いただけましたら、対応させていただきます。
――――――――――――――

なお、日頃の感謝の気持ちとして、ささやかではございますが、お中元のお品を別途お送りさせていただきます。
本来であれば直接ご挨拶にお伺いすべきところ、書面でのご連絡となりますこと、何卒ご容赦ください。

今後とも、皆様のご期待に沿えるよう、より一層努めてまいりますので、
引き続き変わらぬご愛顧を賜りますよう、何卒お願い申し上げます。

厳しい暑さが続く折、皆様におかれましてはくれぐれもご自愛のうえ、
夏をお健やかにお過ごしくださいませ。

今後とも何卒よろしくお願い申し上げます。
```
