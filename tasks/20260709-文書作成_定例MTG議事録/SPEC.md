# SPEC — 定例MTG議事録

## 入力
- 定例MTGの未整形メモ（チャット貼り付け or input/ のテキスト）

## 出力
- `output/20260709-文書作成_定例MTG議事録_{回の日付}.md` として Markdown 1枚

## 議事録フォーマット
```markdown
# 定例MTG 議事録

- 日時: YYYY-MM-DD HH:MM〜HH:MM
- 参加者: 氏名・氏名・氏名

## 決定事項
- 決まったこと

## ToDo（担当・期限）
- [ ] 内容（担当: 氏名 / 期限: YYYY-MM-DD）

## 保留・継続確認
- 様子見・保留の項目

## 次回
- YYYY-MM-DD HH:MM
```

## 整形ルール
- 断片を「決定事項 / ToDo / 保留・継続確認 / 次回」に仕分ける
- ToDo は担当と期限を抽出。期限が曖昧なら今日基準で具体日付に補完する
- 出た人・日時はヘッダーにまとめる
