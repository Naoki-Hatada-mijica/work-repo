```
下記日程にてご調整いただけますと幸いです。
・7月9日（木）12:00～13:00, 18:00～20:00
・7月10日（金）15:00～20:00
```

## サンプル2（終日）

## サンプル3（開始のみ・終了時刻なし）
```
▼面談可能日時
・6月12日（金）15:00～開始
```

## パースで気をつける点
- 1 日に複数枠（カンマ区切り）が来る
- 「終日」表記 → 営業時間の範囲に展開する必要あり
- 「15:00～開始」のように終了時刻がないケース → 想定所要時間で補完 or 要確認
- 日付は「7月9日（木）」「7/1」など表記ゆれあり
