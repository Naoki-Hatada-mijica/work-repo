# work-repo

Claude Code と一緒に日常業務を進めるための、あなた専用の仕事場です。
「優秀だけど記憶が1日で消える同僚」と働くために、大事なことは全部ここのファイルに残します。

## 使い方（これだけ覚えれば OK）

| 言うこと | 起きること |
|---|---|
| 「〇〇をやりたい」「タスク作って」 | `init-task` — タスクフォルダと PLAN/SPEC/TODO/KNOWLEDGE ができる |
| 「今日はここまで。引き継ぎ書いて」 | `handoff` — HANDOFF.md に現状・残タスク・注意点が残る |
| 「終わった」「完了」 | `finish-task` — TODO 確認 → 学びの記録 → GitHub へ保存まで一気に |
| 「スキル化して」 | `make-skill` — 今回のやり方が保存され、次回から一言で呼び出せる |

迷ったら Claude Code に日本語で聞けば教えてくれます。

## 初回セットアップ

このリポジトリを初めて開いたら、Claude Code に **「セットアップして」** と言ってください。
`setup/README.md` の手順に従って、共通ルール（グローバル設定）のインストールと GitHub 連携の確認が自動で進みます。

> **講座ハンズオンの方へ**: セットアップから初タスク・スキル化までを Claude がガイドします。
> **「HANDSON.md を読んで手伝って」** と言ってください（`HANDSON.md` 参照）。

## フォルダ構成

```
work-repo/
├── CLAUDE.md        ← Claude Code への指示書（この仕事場のルール）
├── README.md        ← このファイル（人間向けの説明）
├── HANDSON.md       ← 講座ハンズオン用ガイド（「HANDSON.md を読んで手伝って」で Claude が進行）
├── setup/           ← 初回セットアップ用（完了後に削除してOK）
├── tasks/           ← タスクごとのフォルダ（日付＋タスク名）
│   └── 20260713-経費精算/
│       ├── PLAN.md       ← やりたいことの雑なメモ（あなたが話す → Claude が書く）
│       ├── SPEC.md       ← Claude が質問しながら整理した仕様
│       ├── TODO.md       ← やることリスト（Claude が分解・チェック）
│       ├── KNOWLEDGE.md  ← ハマった点・学び（次回に引き継がれる）
│       ├── input/        ← 受け取った資料（PDF・Excel 等）を入れる場所
│       └── output/       ← 成果物ができる場所
└── .claude/
    ├── skills/      ← スキル（覚えさせた仕事の手順書）
    └── rules/       ← タスク運用の詳細ルール
```

## 大事な考え方

- **雑に話す → Claude が整理する。** きれいな指示を書こうとしなくていい
- **1タスク1会話。** 会話が長くなったら区切って新しく始める（大事なことはファイルに残っているので消えない）
- **繰り返す仕事はスキル化。** 2回目からは一言で終わる
