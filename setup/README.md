# セットアップ（初回のみ）

このフォルダは work-repo の初回セットアップ用です。
ユーザーが「セットアップして」と言ったら、Claude Code は以下の手順を実行してください。

## Claude Code への指示

1. **名前の確認** — ユーザーに名前（漢字フルネーム）を確認する

2. **グローバル設定のインストール**
   - `setup/global-CLAUDE.md` を読み、`{あなたの名前}` をユーザーの名前に置き換える
   - 以下の場所へ `CLAUDE.md` として保存する（`.claude` フォルダがなければ作成する）：
     - Mac: `~/.claude/CLAUDE.md`
     - Windows: `%USERPROFILE%\.claude\CLAUDE.md`
   - 既に `CLAUDE.md` が存在する場合は、上書きせずに内容を見せて「上書きしてよいですか？」と確認する

3. **git の設定確認**
   - `git config user.name` / `git config user.email` を確認し、未設定ならユーザーに聞いて設定する

4. **GitHub 連携の確認**
   - `gh auth status` を実行し、未ログインなら `gh auth login` の手順を1ステップずつ案内する
   - （`gh` コマンド自体がない場合はその旨を報告し、講師に確認するよう伝える）

5. **動作確認と完了報告**
   - グローバル設定が保存されたことを確認する
   - 「セットアップが完了しました。この `setup/` フォルダはもう不要ですが、削除しますか？」と確認し、
     OK なら削除してコミットする

## 補足（人間向け）

- グローバル設定（`~/.claude/CLAUDE.md`）は、この work-repo に限らず **Claude Code を使うすべての作業**で読み込まれる共通ルールです
- work-repo 固有のルール（タスクの進め方・保存の仕方）は、リポジトリ直下の `CLAUDE.md` にあります
