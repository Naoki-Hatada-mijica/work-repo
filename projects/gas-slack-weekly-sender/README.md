# Slack 週次注力情報 手動一括送信ツール

Google Apps Script（GAS）を使って、スプレッドシートからBPパートナー各社のSlackチャンネルへ注力情報を手動一括送信するツールです。

## 特徴

- **手動一括送信** — スプレッドシートのメニューからワンクリックで全社に送信（祝日等の事故防止）
- **個人Slackアカウントから送信** — Botではなく、営業担当者個人のアカウントから投稿
- **パートナー別コンテンツ** — 全体ver / 案件のみ / 人材のみ を企業ごとに出し分け
- **コードブロック対応** — 注力案件・人材リストをコードブロックで見やすく表示
- **URL自動リンク化** — コードブロック内のURLもクリック可能
- **長文自動分割** — Slackの文字数制限を超える場合、区切り線で自動分割して送信
- **スレッド構成** — 親メッセージ → 通常テキスト（挨拶） → コードブロック（詳細）

## メッセージ構成

```
[親メッセージ]  @メンション ⭐ 注力情報のご紹介 7月2週目 ⭐
  └─ [スレッド] 挨拶テキスト（通常テキスト）
  └─ [スレッド] 注力案件・人材リスト（コードブロック）
```

## スプレッドシート構成

| シート名 | 用途 |
|---------|------|
| 送信設定 | パートナー企業・チャンネル・担当者の設定 |
| 全体ver | 案件＋人材の注力情報（全体版） |
| 案件のみ | 案件のみの注力情報 |
| 人材のみ | 人材のみの注力情報 |

### コンテンツシートの書き方

セルA1に `---` 区切りで記入：

```
挨拶テキスト（通常テキストとして送信）
---
注力案件・人材リスト（コードブロックとして送信）
```

- `---` より上 → 通常テキスト
- `---` より下 → コードブロック
- 全体verで長い場合はA2セルに追加コードブロックを記載可

## セットアップ

詳細は [SETUP_GUIDE.md](SETUP_GUIDE.md) を参照。

### 概要

1. [Slack App](https://api.slack.com/apps) を作成し、User Token Scope に `chat:write` を追加
2. アプリをインストールしてユーザートークン（`xoxp-`）を取得（営業2名分）
3. スプレッドシートの「拡張機能」→「Apps Script」で `slack_weekly_sender.gs` を貼り付け
4. スプレッドシートを再読み込みし、メニュー「Slack送信」→「トークン登録」で設定
5. 「テスト送信」で動作確認

## 毎週の運用フロー

1. コンテンツシート（全体ver / 案件のみ / 人材のみ）を更新
2. 「Slack送信」→「プレビュー」で内容確認
3. 「Slack送信」→「一括送信」で全社に送信
4. 送信結果ダイアログで成功/エラーを確認
